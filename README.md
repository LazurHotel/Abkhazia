# ☕ Кофе-принт — сайт кофейни и копицентра

Сайт нового заведения **«Кофе-принт»** — кофейня и копировальный центр в одном месте.
Объединяет лучшее из кофейного и копи-сервиса: можно распечатать диплом, курсовую
или семейные фото и тут же выпить свежий кофе.

**Целевая аудитория:** студенты, молодые семьи с детьми и школьниками.

## Что внутри

### Витрина для клиентов (`index.html`)
- **Меню кофейни** с табами: кофе, холодное, детское меню, перекус (цены, описания).
- **Услуги печати** — 9 карточек (печать, копирование, скан, переплёт, ламинирование,
  фото на документы, широкоформат, печать фото, сувенирка) с кнопкой «Рассчитать».
- **Живой калькулятор стоимости** печати: выбор услуги и параметров, автоматический
  пересчёт, студенческая скидка −10%.
- **Онлайн-заказ** с прикреплением файлов, выбором самовывоза/доставки и временем.
- **Программа лояльности** — виртуальная карта «6-й кофе бесплатно» с интерактивными штампами.
- **Акции для ЦА**, блок удобств (Wi-Fi, розетки, детская зона), FAQ, контакты, форма связи.

### Кабинет управляющего (`admin.html`)
- **KPI:** всего заказов, выручка, средний чек, заказы за сегодня, в работе, гости лояльности.
- **Таблица заказов** с фильтрами и переключением статусов
  (Новый → В работе → Готов → Выдан).
- **Выручка по услугам** (диаграмма), **очередь печати**, список участников лояльности.
- **Экспорт заказов в CSV**, управление демо-данными.

Заказы с витрины и карты лояльности сохраняются в `localStorage` и сразу видны в кабинете.

## Структура

```
index.html      — витрина для клиентов
admin.html      — кабинет управляющего
css/style.css   — стили витрины
css/admin.css   — стили кабинета
js/data.js      — меню, услуги, цены, акции, контакты (window.CP)
js/store.js     — хранилище заказов и лояльности в localStorage (window.CPStore)
js/app.js       — логика витрины (меню, калькулятор, заказы, лояльность)
js/admin.js     — логика кабинета управляющего
```

## Запуск

Статический сайт без сборки. Откройте `index.html` в браузере или поднимите
локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Кабинет управляющего доступен по ссылке в подвале или напрямую: `/admin.html`.
